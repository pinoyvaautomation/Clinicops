# ClinicOps Demo Call Script

Use this for `10-15 minute` founder demos with clinic owners or managers.

## 1. Opening

Hi, thanks for taking the time.

I’ll walk you through ClinicOps quickly so you can see whether it fits your clinic.

The goal is simple:

- easier booking
- less front-desk back-and-forth
- cleaner patient intake
- a better operations view for the clinic owner

## 2. Discovery Questions

Ask these first:

- What type of clinic do you run?
- How many staff usually handle appointments?
- How are bookings managed today?
- Do patients mostly book by phone, chat, or website?
- What is the biggest admin pain point right now?

## 3. Frame The Demo

Based on what you shared, I’ll show you the parts that matter most:

- public booking
- intake and consent
- staff workflow
- self-service appointment changes
- Premium tools if relevant

## 4. Public Booking Walkthrough

Show:

- clinic booking page
- selecting a service
- booking an appointment
- patient intake fields

Say:

This gives patients a direct online booking path instead of relying only on calls or chat.

## 5. Self-Service Appointment Changes

Show:

- appointment confirmation
- secure manage link
- cancel or reschedule flow

Say:

This reduces front-desk back-and-forth because patients can make changes themselves.

## 6. Staff Portal Walkthrough

Show:

- dashboard
- appointments
- patients
- staff
- services

Say:

The clinic team gets one workspace instead of splitting this across multiple tools or manual tracking.

## 7. Premium Walkthrough

Show only if relevant:

- messaging
- waitlist
- reminders
- branding

Say:

Free is designed to help clinics get started.
Premium is for clinics that want stronger day-to-day operations and staff efficiency.

## 8. Founder Offer

Say:

Right now we are opening a founder offer for early clinics:

- `Premium at $19/month`
- locked for `12 months`
- available to the first `50 clinics`

You can also start with Free and upgrade later.

## 9. Objection Handling

### If they say "We already use something"

That makes sense.

ClinicOps is best for clinics that want a lighter booking and operations workflow without a heavy enterprise system.

### If they say "We are too small"

That is actually where ClinicOps fits well.

Free lets smaller clinics start without a large commitment, and Premium becomes useful as the clinic gets busier.

### If they say "We need time"

No problem.

I can send a short summary, demo link, and the founder offer so you can review it with your team.

## 10. Close

Ask one clear next-step question:

- Would you like to start with Free?
- Would you like help activating the founder Premium offer?
- Should I send the signup link and short recap?

## 11. After-Call Follow-Up

Send:

- signup link
- short recap
- founder offer details
- next step
