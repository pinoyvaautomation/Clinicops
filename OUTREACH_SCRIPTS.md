# ClinicOps Outreach Scripts

Use these for Facebook DM, email, or LinkedIn outreach.

Keep messages short. Do not send a wall of text.

## 1. Cold DM

Hi [Name], I’m reaching out because we built `ClinicOps` for small clinics that still handle a lot of bookings and follow-up manually.

It helps clinics manage:

- online booking
- intake forms
- self-service appointment changes
- staff workflow in one dashboard

We are opening a founder offer for the first `50 clinics`.

If you want, I can send a short demo link.

## 2. Follow-Up DM

Hi [Name], following up in case this is useful for your clinic.

ClinicOps is designed for small teams that want to reduce front-desk back-and-forth and make booking easier for patients.

If you want, I can send:

- a 1-minute demo
- or a quick walkthrough link

## 3. After Reply

Thanks. To see if it fits, can I ask:

- what type of clinic do you run?
- how many staff usually handle appointments?
- are bookings mostly done by phone, Messenger, or website today?

## 4. Demo Invite

Based on that, I think ClinicOps could help.

The easiest next step is a short demo so you can see:

- public booking
- patient intake
- appointment handling
- Premium tools like messaging and waitlist

Would you like a demo link or a quick walkthrough call?
